# Keyword + backlink playbook (scrubbed)

Last scrub: 2026-07-23. Machine-readable targets: `src/data/keywords.ts`.

## What works (keep doing)

| Channel | Tactic | Why |
|---------|--------|-----|
| Organic | Literacy intents: COA, RUO, evidence tiers, “{compound} evidence” | YMYL-safe; vendors spam “buy” SERPs we skip |
| Organic | Tool + explainer pairs (recon calculator) | High engagement; coach/creator links |
| Organic | Compound hubs with honest evidence badges | Differentiates from miracle pages |
| AEO | Definitional first sentences + glossary + `llms.txt` | Answer engines pull clean defs |
| Backlinks | Resource-page / LibGuide / .edu lists | Highest trust-per-link for education |
| Backlinks | Journalism citations (RUO, GLP-1 media literacy) | Referral + brand entity |
| Backlinks | Newsletter swaps of evidence-tier explainers | Warm audiences |
| Commerce | Bac water / syringe accessories queries | Matches inventory; cleaner than peptide SKUs |

## What does not work (banned)

- “Buy BPC-157” / dosing protocol content
- PBN, paid medical fake citations, link farms
- Thin duplicate compound pages without sources
- Fake clinician-review badges
- Mass unpersonalized outreach blasts
